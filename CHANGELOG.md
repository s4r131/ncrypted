# Changelog

## 1.0.0

- Initial production-ready app shell for `ncrypted`.
- Implemented end-to-end encrypted file send/open flow with `.ncry` packages.
- Added identity and contacts management with QR workflows.
- Added advanced security settings (RSA size/exponent, verified-sender policy).
- Added theme customization, system-following appearance option, and options screen.
- Added terminal easter-egg mode at max security profile.
- Hardened package parsing and modernized cleanup/analyzer fixes.
